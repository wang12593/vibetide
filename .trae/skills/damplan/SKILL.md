# Damplan - 企业媒体资产管理系统产品规划

## 概述

本Skill用于规划企业媒体资产管理系统（MAM/DAM）产品，基于华栖云公司技术能力和市场定位，输出专业的产品规划报告。

## 产品基本信息

- **产品名**: Vdam
- **产品类型**: 面向企业的媒体资产管理（MAM）系统、数字资产管理(DAM)系统
- **产品定价**: 30万元
- **目标市场**: 企业市场（石油、电力、烟草、金融等）、政府市场（博物馆、文旅、公安、消防、教委等）、媒体行业

## 华栖云公司核心能力

- **愿景**: 让每一段视频都产生价值
- **核心技术**:
  1. 云端超高清视频"汇聚、处理、识别、理解、重建"全链路业务能力
  2. 华生智媒大模型能力：面向视音频处理场景的融合模型，专注处理非结构化数据
  3. AI native的专业系统、视音频工具、智能体

## 数据收集方法（确保完整性）

为保证数据收集的完整性和可靠性，本Skill采用以下数据收集策略：

### 核心原则

1. **多关键词组合搜索**
   - 不依赖单一关键词，使用多个相关关键词组合
   - 覆盖不同角度和场景，确保数据源充足

2. **分批执行与实时保存**
   - 将大数据量目标拆分为多批次执行
   - 每批次完成后立即追加保存，避免数据丢失
   - 使用追加模式（append）而非覆盖模式

3. **实时验证与补充**
   - 每批次完成后统计当前数据条数
   - 最终验证是否达到目标数量
   - 不足则自动补充搜索直至达标

4. **进度透明反馈**
   - 每批次完成后向用户报告进度
   - 让用户了解数据收集的实时状态

| "2-5" / "2~5" | Step 2,3,4,5 | 执行指定范围步骤，连续执行不中断 |
| "否" / "跳过" / "skip" | 无 | 跳过数据获取，进入写作阶段 |

**重要约束**：
1. 必须等待用户明确回复后才能执行数据获取
