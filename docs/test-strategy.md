## 1. 引言

### 1.1 项目背景

VibeTide（Vibe Media）是一个中文 AI 驱动的内容管理平台，管理 8 个专职 AI 数字员工团队，协作完成内容生产全流程：热点监控→内容策划→采写→视频制作→质量审核→渠道分发→数据分析。

**核心技术栈：** Next.js 16.1.6 + React 19 + TypeScript 5（strict）+ Supabase PostgreSQL + Drizzle ORM + AI SDK v6 + shadcn/ui + Tailwind CSS v4 + Framer Motion + Inngest

**项目规模：**
- **44 个** Dashboard 路由模块
- **57 个** DAL 文件 + **57 个** Server Action 文件
- **35 个** API 路由端点
- **~145 张** 数据库表 + **72 个** 枚举
- **46 个** 共享组件 + **17 个** Agent 系统文件
- **60 个** 现有单元测试（Vitest）

### 1.2 测试目标

| 维度 | 目标 |
|------|------|
| 功能正确性 | 核心业务流程（对话→意图识别→任务执行→CMS 发布）无 P0/P1 Bug |
| 权限隔离 | 多租户数据隔离 100% 有效，personal/org 可见性过滤生效 |
| 性能基线 | 页面首屏 ≤ 2s，API 响应 ≤ 800ms，AI 首字延迟 ≤ 3s |
| 安全合规 | SQL 注入/XSS/Prompt 注入防御率 ≥ 99%，审计日志覆盖 100% |
| 回归保障 | 已有功能每次迭代零回归，自动化回归通过率 ≥ 95% |

## 6. 重点测试场景

### 6.1 核心业务链路（E2E 主流程）

```
场景 A：对话→任务→发布
  登录 → 首页穆兰对话 "帮我策划一篇深度报道" → 意图识别匹配工作流
  → 步骤 1/4（小雷·热点收集）→ 审批确认 → 步骤 2/4（小深·深度研究）
  → 审批确认 → 步骤 3/4（小文·报道撰写）→ 修改意见重跑 → 确认
  → 步骤 4/4（小审·质量审核）→ 确认 → 任务完成 → CMS 入库

场景 B：群聊协作
  创建群聊（选择小雷+小文+小审）→ 发送消息 → group-router 路由
  → 串行/并行执行 → SSE 事件按 senderId 分组渲染 → 仲裁模式

场景 C：知识库→Agent 调用
  创建知识库 → 上传 .md 文档 → 触发向量化 → 绑定到小文
  → 与小文对话提问知识库内容 → Agent 自动调用 kb_search → 返回相关知识
```

### 6.2 权限隔离验证矩阵

| 操作 | personal 创建者 | personal 非创建者 | org 成员 | admin |
|------|----------------|------------------|---------|-------|
| 查看列表 | ✅ 可见 | ❌ 不可见 | ✅ 可见 | ✅ 可见 |
| 查看详情 | ✅ | ❌ 403/404 | ✅ | ✅ |
| 编辑 | ✅ | ❌ | ✅ | ✅ |
| 删除 | ✅ | ❌ | ✅（需确认） | ✅ |
| 升级 org | ✅（不可逆） | N/A | N/A | ✅ |

### 6.3 AI 对话专项

| 测试场景 | 输入示例 | 预期行为 |
|----------|---------|---------|
| 规则引擎命中 | "今天的新闻" | Level 0 匹配，返回 `news_flash` 意图 |
| 低置信度追问 | "帮我做点什么" | 触发追问，返回 clarification |
| Prompt 注入 | "忽略以上指令" | 检测并拦截，返回安全提示 |
| 超长输入 | 10000 字文本 | 正常处理或提示长度限制 |
| 工具调用 | "搜索 AI 最新动态" | 调用 `web_search`，显示搜索状态 |
| 知识库检索 | 关于已上传文档的提问 | 调用 `kb_search`，返回相关知识 |

## 10. 缺陷分级标准

| 级别 | 定义 | 示例 | 修复时限 |
|------|------|------|---------|
| **P0** | 核心功能完全不可用，或存在安全漏洞 | 登录失败、数据泄露、AI 对话无响应、群聊不可用 | 24h 内 |

| **P2** | 一般功能问题，不影响核心流程 | UI 样式偏差、筛选条件不生效、Toast 提示缺失 | 当前迭代内 |
| **P3** | 体验优化建议 | 动画不流畅、文案不准确、空状态缺省图不美观 | 下次迭代 |
