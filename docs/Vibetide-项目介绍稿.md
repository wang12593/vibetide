# Vibetide（Vibe Media）— 项目介绍稿

> **数智全媒平台 · AI内容生产新范式**

---

## 一、项目定位

**Vibe Media（Vibetide）** 是一款面向中文媒体行业的 **多智能体协同开放式内容生产引擎**。它组建了一支由 8 位专业 AI 数字员工组成的虚拟编辑部，通过智能体自主协作，覆盖从热点捕获到全渠道分发的完整内容生产链路。

**核心价值主张：**

| 维度 | 传统媒体团队 | Vibetide AI 团队 |
|------|-------------|-----------------|
| 热点响应速度 | 人工监控，小时级 | AI 7×24 监控，5 秒锁定 |
| 全流程耗时 | ~4 小时 | ~3 分钟 |
| 团队规模 | 数十人编辑部 | 8 位 AI 员工 |
| 运营成本 | 高（人力+管理） | 低（SaaS 订阅） |
| 产出稳定性 | 依赖个人状态 | AI 持续稳定输出 |

**品牌承诺：** 无需信用卡 · 免费使用 · 即刻上手

### 2.2 技能体系

每位员工配备 **28+ 内置技能**，分为 6 大类别：

- **信息感知（info_perception）** — 全网搜索、热榜聚合、社交聆听
- **信息处理（info_processing）** — 内容生成、风格改写、质量审核
- **系统互操作（system_interop）** — 发布策略、媒资检索、知识检索
- **自动化（automation）** — 趋势监控、任务规划、批量生产
- **通信（communication）** — 受众分析、多语翻译
- **多模态（multimodal）** — 视频剪辑方案、封面生成、音频方案

技能支持动态绑定、自由组合，员工可根据任务需求灵活配置技能包。

## 四、实战应用场景

### 场景 1：突发快讯（~5 分钟）

```
小雷(热点捕获) → 小文(快讯撰写) → 小审(质量审核) → 小发(多平台抢发)
```

比同行快 **10 倍**，第一时间占领信息高地。

### 场景 2：深度专题（~30 分钟）

```
小雷 → 小策(AI策划角度) → 小资(资料收集) → 小文(深度撰写) → 小简(配图配视频) → 小审(审核发布)
```

6 位员工协同作战，产出高质量深度报道。

### 场景 3：日常运营（每日自动）

```
小雷(每日选题) → 小策(策划) → 小文(批量创作) → 小发(定时发布) → 小数(数据复盘)
```

每日自动生成选题计划、批量创作、定时发布、数据回收复盘。

### 场景 4：赛事/会展自动化

```
监控赛事 → 自动裁剪 → 生成报道 → 多平台发布
```

体育赛事、会议、节庆、展览等场景自动化报道。

## 八、设计系统

### Glass UI 毛玻璃风格

产品采用统一的 **毛玻璃（Glass）设计语言**，核心原则：

- **GlassCard** — 所有卡片统一使用毛玻璃效果
- **Button** — 液态玻璃半透明天空蓝风格，6 种变体
- **无原生标签** — ESLint 强制禁止 `<button>/<input>/<select>/<textarea>`，必须使用组件库
- **无边框按钮/标签** — 所有可交互元素不带边框
- **中文界面** — 所有 UI 文本使用简体中文

### 组件体系

```
UI 基元层     →  shadcn/ui (25+ 组件)     → Button / Input / Select / Tabs / Dialog ...
共享业务层    →  shared/ (45 个组件)        → GlassCard / DataTable / EmployeeAvatar / SearchInput ...
图表层        →  charts/                    → Area / Bar / Donut / Gauge / Radar / HeatCurve
布局层        →  layout/ (4 个组件)         → AppSidebar / Topbar / DashboardShell / RightSidebar
```

## 十二、总结

**Vibetide（Vibe Media）** 不仅仅是一个内容管理系统，而是一个 **AI 原生的内容生产操作系统**。它通过以下三个层面重新定义了媒体内容生产：

1. **组织层面** — 用 8 位 AI 数字员工替代传统数十人编辑部，成本降低一个数量级
2. **流程层面** — 从热点捕获到全渠道分发的全链路自动化，4 小时压缩到 3 分钟
3. **智能层面** — 7 层 Prompt 组装 + 17+ 工具注册 + 知识库向量检索，让 AI 真正"理解"媒体业务

**一句话概括：** 让 AI 团队，成为你的内容引擎。
